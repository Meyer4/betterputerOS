# Contributing to BetterPuterOS

Thank you for helping build a better OS!

## How to Contribute

- **Ideas & Feedback:** Open a GitHub Discussion or Issue.
- **Code:** Fork, branch, and submit PRs.
- **Design:** Contribute themes, icons, and UX ideas.
- **Docs:** Help write or translate docs in `/docs`.
- **Community:** Join us on Discord, share, and invite others!

## Code of Conduct

Be kind, inclusive, and constructive. We build together.
